# Llava-CLI-Sample

Llava をローカルで実行するためのサンプル。

モデルは以下のものを使用する。
https://huggingface.co/mys/ggml_llava-v1.5-7b/tree/main

### 使い方

実行スクリプトを実行する。

環境構築（ライブラリのインストール、モデルのダウンロード、サンプル画像のダウンロード）が行われ、最終的にレスポンスが返ってくる。

```bash
$ sh run.sh
```

入力画像やテキストを変更したい場合は、`run.sh` 内の以下の箇所を変更する。

```sh:run.sh
INPUT_TEXT="この画像を詳細に説明してください。回答は日本語にしてください。"
IMAGE="sample.jpg"
```
